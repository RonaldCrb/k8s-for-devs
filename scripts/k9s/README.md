# K9S INSTALLATION SCRIPT FOR LINUX

## USAGE
run the ./scripts/k9s/install.sh script to install k9s locally in your machine

## AUXILIARY DOCS

this is the main repository for k9s, please make sure you download the latest version from the releases page you can edit the `install.sh` file to add the current version at the top
